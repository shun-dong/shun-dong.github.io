When a =A b is inhabited, we say that a and b are (propositionally) equal

However, in type theory there is also a need for an equality judgment, existing at the same level as the judgment “x : A”. This is called judgmental equality or definitional equality
….in particular, it is algorithmically decidable (though the algorithm is necessarily meta-theoretic, not internal to the theory).

reflexivity: a:: A -> a = a
for a: A := b, we have refl$_a$: a =$_A$ b

[[Homology_Type_Theory.pdf#page=60&selection=227,0,227,30&color=note|Indiscernibility_of_identicals]]: There exists a function f(x: A, y: A, p: x$=_{A}$y): C(x) -> C(y) s.t. f(x, x, refl$_{x}$) = id
f是transport函数, 实现通过语义相等在语法上将C(x)替换为C(y), 如果p不是refl的话会有其他方法

(x:: A, y::A, x=$_A$y) is called [[Homology_Type_Theory.pdf#page=63&selection=225,1,226,15&color=note|free_path_space]]
[[Homology_Type_Theory.pdf#page=61&selection=45,0,45,14&color=note|Path_induction]]: Given a family C(x: A, y: A, p: x$=_{A}$y): Type 实际上参数是标准的相等(路径)的配置, and a function c(x: A): C(x, x, refl$_{x}$)这里是将refl看作最基础的路径, 通过最基础的路径得到了C中的一项, there exits a function f(x: A, y: A, p: x$=_{A}$y):C(x, y, p) s.t. f(x, x, refl$_{x}$) = c(x)然后可以得到任意路径对应的项, 类似于归纳法, 这些非refl的元素构成了hott的丰富结构
J:: (C::(x:: A-> y:: A-> p: x$=_{A}$y-> Type)->(x:: A->C(x, x, refl$_{x}$))->(x::A-> y::A-> p: x$=_{A}$y->C(x, y, p))),
J(C, c, x, x, refl$_{x}$) = c(x)
这里如果固定x就得到了based path induction, 但注意不能固定两个端点, 这样直观来讲路线可能缩不回去

Definition of Homotopy type theory

[!PDF|] [[Homology_Type_Theory.pdf#page=71&selection=4,0,5,64|Homology_Type_Theory,_p.59]]
The central new idea in homotopy type theory is that types can be regarded as spaces in homotopy theory, or higher-dimensional groupoids in category theory.

[[Homology_Type_Theory.pdf#page=71&selection=135,0,135,8&color=note|homotopy]] h: f g: (X -> Y), h: X * [0,1] -> Y -> h( _ , 0) = f, h( _ , 1) = g
for paths form y1 to y2 (i.e. f g: ([0, 1] -> Y), f(0) = g(0) = y1, f(1) = g(1) = y2), we also require h(0, _ ) = y1, h(1, _ ) = y2. h is called 2-dimensional path between paths
in specific, for loops at y, we requires h(0, _ ) = h(1, _ ) =y. It’s called based homotopy

p, q has a homotopy is an equivalence equation
[[Homology_Type_Theory.pdf#page=72&selection=10,0,10,17&color=note|fundamental_group]]: homotopy equivalence classes of loops at some point y
NB 连通空间内不同点的基本群自然同构
eg. 一个普通的圆环（S1）的基本群是整数加群，环路绕圆环多少圈（正绕/反绕）就是其元素。
homotopy equivalent X Y: f: (X -> Y), g: (Y -> X), f @ g and g @ f are homotopic to the identity
homotopy equivalent spaces have isomorphic fundamental groups
NB 反之不成立

his infinite tower of points, paths, homotopies, homotopies between homotopies… equipped with algebraic operations such as the fundamental group, is an instance of (week, up to homotopy at the next level) [[Homology_Type_Theory.pdf#page=72&selection=34,1,36,9&color=note|∞-groupoid]], called fundamental ∞-groupoid
goupoid(群胚): 箭头都可逆的范畴
a morphism at level k is called a k-morphism
homotopy hypothesis/theorem: the fundamental ∞-groupoid construction is adjoint to the geometric realization of an ∞-groupoid as a space

path from x to y. x: A, y: A, p: x $=_A$ y
we call x its start point and y its end point. Two paths p, q : x =A y with the same start and end point are said to be parallel
