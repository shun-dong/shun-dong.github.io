色彩

色相(Hue)

颜色的名字

色盘:光谱红紫相连
可以分解到两个方向, 色温(黄-蓝), 色调(绿-洋红)

饱和度(Saturation)

纯度，颜色的纯净程度

明度/灰度(Lightness/Brightness )

Gamma值(调整Gamma函数)
从暗到明的灰度阶
明度三大调子：高调、中调、低调

影调（面积-明度分布）

高调、中调、低调曝光
长调(现实),中调,短调
软调(分布在中间),硬调(分布在两边)对比度
全长调:硬长调

$$
H=\left{\begin{array}{ll}
0 & \text { if } \max =\min \
60^{\circ} \times\left(\frac{G^{\prime}-B^{\prime}}{\max -\min }\right)+0^{\circ} & \text { if } R^{\prime}=\max \
60^{\circ} \times\left(\frac{B^{\prime}-R^{\prime}}{\max -\min }\right)+120^{\circ} & \text { if } G^{\prime}=\max \
60^{\circ} \times\left(\frac{R^{\prime}-G^{\prime}}{\max -\min }\right)+240^{\circ} & \text { if } B^{\prime}=\max
\end{array}\right.
$$
$$

S=\frac{\max -\min}{\max}

$$
$$

L=\max (R’ , G’ , B’)

$$
H相当于颜色最大值不变的情况下RGB走一圈
S相当于颜色间的对比, S越高对比越强烈, 相当于减少次要通道
L相当于色彩的总量, 单独改变L相当于等比改变RGB三个通道

七大色调

把明度和饱和度放在一起得到七大色调(其实和色调不是一个概念)
白色调、粉色调、明色调、鲜色调
灰色调、深色调
黑色调

明度高的地方▶高调
白色调

  • 饱和度很低，明度很高
  • 素雅、文艺、干净、极简
  • 乏味、单调
    粉色调
  • 饱和度要比白色调高一些，明度要比白色调低一些
  • 清秀、天真、可爱
  • 软弱无力，缺少力量感
    明色调
  • 相对的要鲜艳一些
  • 年轻、活力、阳光、清爽
  • 比较的廉价
    鲜色调
  • 明度和饱和度都非常的高，给人的视觉冲击力很强
  • 刺激、活力、强力
  • 较为生硬和廉价
    灰的地方▶中调
    ==灰色调==
  • 高级灰，明度不高，饱和度也不高，看上去有高级感
  • 品质、高端、奢华、生态
  • 缺少活力。
    深色调
  • 深色调深色调相比起灰色调饱和度会更高一些，看上去会更加的深邃
  • 成熟、稳重、大气
  • 比较压抑、阴暗，不透气
    明度低的地方▶低调
    黑色调
  • 接近于黑色的色调，饱和度和明度都很低
  • 神秘、力量、厚重
  • 抑郁、压抑

混色

  • 色温：控制黄-蓝平衡。低色温增强黄色，高色温增强蓝色。
  • 色调：控制绿-洋红平衡。调整可中和环境光的偏色（如绿色偏移时增加洋红）

lut

Look-Up Table
把“输入数值 → 输出数值”的对应关系，提前算好，存在一张表里，以后只要“查表”就能快速完成复杂转换。

  • 技术lut：将专业摄影机拍摄的 LOG 格式（低对比度、灰片）快速还原为正常色彩
  • 风格lut：类似滤镜
